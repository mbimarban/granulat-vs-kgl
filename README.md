# Granulat vs KGL

Porównawcza analiza finansowa Granulat i KGL.

## Zawartość
- porównanie marży brutto i marży operacyjnej
- analiza cash flow i jakości wyniku
- porównanie struktury kosztów
- analiza red flags i różnic biznesowych
- wersja PDF do pobrania
